# PhysicsEngine | 物理引擎

  这是一个用C++写的简单的刚体物理引擎，在我的仓库的2d physics engine的基础上完善了代码和功能。
  
## 实现的功能  
  其中基本的物体类型有多边形与圆形  
  实现了基本物体类型间的碰撞检测（SAT算法）  
  能够获取各种物体类型间的接触点  
  实现了基于冲量的碰撞求解（包括旋转与摩擦）
